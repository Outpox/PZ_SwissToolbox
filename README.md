PZ_SwissToolbox
===============

Mod for project Zomboid "Build 23" (!) that combine a hammer, a saw and a screwdriver in one lighter tool.

Version
=======
0.3

Features
========
- Combination possible
- Custom texture
- Dismantling is working (gives back all the item)
